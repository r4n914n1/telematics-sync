# Orbit Board Web App

Orbit Board is a small responsive web app for managing daily tasks.

## Features

- Add tasks
- Mark tasks as done/undone
- Filter by all, active, and done
- Delete individual tasks
- Clear all completed tasks
- Persistent data using browser localStorage

## Run

1. Open `index.html` in your browser.
2. Or use VS Code Live Server on this folder for auto-reload.

## Project Files

- `index.html` - App structure
- `styles.css` - Visual design, animations, responsive layout
- `app.js` - Task logic and localStorage integration
